Análise e Desenvolvimento de Sistemas (ADS)

O curso de ADS da Fatec Sorocaba prepara profissionais para atuar no desenvolvimento de software, com foco em qualidade, segurança e usabilidade.

Principais disciplinas:

  • Algoritmos e Estrutura de Dados
  • Programação Web
  • Banco de Dados
  • Engenharia de Software
  • Interface Humano-Computador
  • Programação Orientada a Objetos

Duração: 3 anos
Turno: Noturno
Modalidade: Presencial
